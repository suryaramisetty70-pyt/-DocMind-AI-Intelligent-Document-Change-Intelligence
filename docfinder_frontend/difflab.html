Frequently Asked Questions

What file formats does DiffLab support?

DiffLab supports plain text files (.txt, .md, code files), PDF documents (.pdf), Excel spreadsheets (.xlsx, .xls), and CSV tables (.csv). All comparisons are fully optimized for their respective schemas.

How secure is my data on DiffLab?

Extremely secure. All uploads are fully encrypted using AES-256 both in transit and at rest. If you are using our local desktop node, your files never leave your computer, processing everything locally offline.

How does the AI semantic compare work?

Instead of just comparing text characters, our AI parses the meaning and context of the changes. For example, it detects if a clause was rephrased, if terms were shifted, or if elements in an image/PDF were visually rearranged.

Can I collaborate with my team in real time?

Yes. Our Live Collaboration rooms let you create workspace rooms and sync text checks in real-time, complete with a live similarity indicator and active participant counter.
